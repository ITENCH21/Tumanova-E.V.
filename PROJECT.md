# resume.html — Резюме Тумановой Елены Викторовны

## Что это

Одностраничное интерактивное HTML-резюме. Один файл, без зависимостей, открывается в любом браузере.

---

## Функциональность

| Кнопка | Действие |
|--------|----------|
| RU / EN | Переключение языка (все тексты дублированы в `data-ru` / `data-en`) |
| ☀️ / 🌙 | Светлая / тёмная тема (сохраняется в localStorage) |
| PDF | `window.print()` — браузерная печать в PDF |

---

## Структура файла

```
resume.html
│
├── <style>          — весь CSS (~400 строк)
│   ├── CSS-переменные :root (тёмная тема по умолчанию)
│   ├── [data-theme="light"] — светлая тема
│   ├── .controls    — плавающая панель кнопок сверху
│   ├── .layout      — двухколоночная сетка (260px + 1fr)
│   ├── .sidebar     — левая колонка
│   ├── .hero        — карточка-заголовок
│   ├── .banner      — баннер "Открыта к предложениям"
│   ├── .card        — секции опыта и образования
│   └── .timeline    — таймлайн опыта работы
│
├── <body>
│   ├── <canvas id="starsCanvas">  — анимированный фон
│   ├── <nav class="controls">     — кнопки управления
│   └── <div class="page">
│       ├── <aside class="sidebar">
│       │   ├── Фото (загрузка через FileReader)
│       │   ├── Контакты
│       │   ├── Ключевые навыки (progress bars)
│       │   ├── Сильные стороны (chips)
│       │   └── Инструменты (tags)
│       └── <main class="main">
│           ├── .hero       — имя + должность + о себе
│           ├── .banner     — блок открытости к работе
│           ├── .card       — опыт работы (таймлайн)
│           └── .card       — образование
│
└── <script>         — весь JS
    ├── Загрузка фото
    ├── setTheme(t)  — переключение темы
    ├── setLang(l)   — переключение языка
    ├── animateBars()— анимация progress bars при загрузке
    └── Canvas IIFE  — анимированный фон (звёзды / пух + лучи)
```

## Анимированный фон (Canvas)

Логика в IIFE-функции внизу `<script>`. Определяет тему через `isDark()` и рисует разные эффекты:

**Тёмная тема — звёзды:**
- 180 точек, случайный drift, мерцание через dalpha

**Светлая тема — тополиный пух + солнечные лучи:**
- **Пух:** 65 пушинок, каждая с 6–9 ворсинками; дрейфуют вверх, покачиваются синусоидой, медленно вращаются
- **Лучи:** 9 золотых лучей из левого верхнего угла (0,0); треугольные градиентные формы, медленно пульсируют

При смене темы `MutationObserver` пересоздаёт частицы пуха.

## Контент — как редактировать

### Личные данные (сайдбар)
```html
<!-- Имя -->
<div class="photo-name">Туманова<br>Елена Викторовна</div>

<!-- Должность -->
<div class="photo-role" data-ru="..." data-en="...">...</div>

<!-- Телефон -->
<span>+7 (962) 684-90-11</span>

<!-- Email -->
<span>Kargina_2015@mail.ru</span>

<!-- Город -->
<span data-ru="Санкт-Петербург" data-en="Saint Petersburg">...</span>
```

### Навыки (прогресс-бары)
```html
<div class="sb-skill-label">
  <span data-ru="Название" data-en="Name">Название</span>
  <span class="sb-skill-pct">95%</span>
</div>
<div class="sb-track">
  <div class="sb-fill" data-w="95"></div>  <!-- data-w = процент -->
</div>
```

### Опыт работы — один блок
```html
<div class="tl-item">
  <div class="tl-left"><div class="tl-dot"></div></div>
  <div class="tl-right">
    <div class="tl-header">
      <span class="tl-cat" data-ru="Категория" data-en="Category">Категория</span>
      <span class="tl-title" data-ru="Должность" data-en="Position">Должность</span>
      <span class="tl-year">2022–н.в.</span>
    </div>
    <div class="tl-place" data-ru="Компания" data-en="Company">Компания</div>
    <div class="tl-desc" data-ru="Описание RU" data-en="Description EN">Описание RU</div>
  </div>
</div>
```

### Двуязычность
Любой элемент с `data-ru` и `data-en` автоматически переключается функцией `setLang()`.
Текст внутри тега = значение по умолчанию (RU).

## Быстрые правки

**Сменить цвет акцента** → поменять `--accent` и `--accent2` в `:root` и `[data-theme="light"]`

**Добавить место работы** → скопировать блок `.tl-item` в таймлайне, вставить выше последнего

**Изменить базовый размер шрифта** → `html { font-size: 17px; }` (сейчас 17px)

**Отключить анимацию фона** → удалить тег `<canvas id="starsCanvas">` и весь Canvas IIFE в `<script>`
